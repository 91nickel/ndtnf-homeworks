**Задание 2.**

Создать модуль для работы с комментариями:

- Создать модель BookCommentModel.
- Создать сервис BookCommentsService для работы с BookCommentModel.

Модель BookCommentModel должна хранить: 

```js
{
    id: "number"

}

```
